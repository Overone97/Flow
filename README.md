# Flow

Prototype de démo artistique 3D web avec ambiance fantasy douce.

## Version
0.0.1 - Prototype initial

## Lancer le projet
Ouvrir `index.html` dans un navigateur moderne.

```bash
# Ou avec un serveur local
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```

## Contrôles
- WASD / ZQSD : Déplacement
- Souris : Rotation caméra (cliquer pour capturer)
- Espace : Saut

## Structure
- `index.html` - Application complète (Three.js via CDN)
